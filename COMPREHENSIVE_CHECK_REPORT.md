### 2.1 crypto.randomUUID 兼容性问题
**问题描述：**
- 部分浏览器不支持 `crypto.randomUUID()` 方法
- 导致测评提交时出现错误

**修复方案：**
- 文件：`/workspace/src/store/assessmentStateMachine.ts`
- 添加了兼容性检查和降级方案

**修复代码：**
```typescript
const generateId = () => {
  if (typeof crypto !== 'undefined' && crypto.randomUUID) {
    return crypto.randomUUID()
  }
  return `${Date.now()}-${Math.random().toString(36).substr(2, 9)}`
}
```
