# 心镜 MindMirror - Web 跨平台版本

这是一个使用 Vue 3 开发的跨平台心理测评 Web 应用，可以在浏览器、手机、平板上完美运行。

## 🚀 快速启动

### 方式一：直接打开（推荐）

直接用浏览器打开 `index.html` 文件即可使用。

### 方式二：本地服务器

如果你有 Python，可以快速启动一个本地服务器：

```bash
# Python 3
python3 -m http.server 8080

# Python 2
python -m SimpleHTTPServer 8080
```

然后在浏览器访问：http://localhost:8080

### 方式三：使用 VS Code

在 VS Code 中安装 "Live Server" 插件，右键点击 `index.html` → "Open with Live Server"。

## ✨ 功能特性

- 📋 **测评库**：多种心理测评工具
- 🧠 **测评流程**：完整的测评体验
- 📊 **结果展示**：详细的分析和解读

- 📚 **心理知识**：学习心理学相关知识

- **Vue 3**：渐进式 JavaScript 框架
