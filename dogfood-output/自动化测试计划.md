## 测试用例

### TC-001: 完整测评流程
**前置条件:** 用户已访问首页
**测试步骤:**
1. 点击"开始测评"或类似按钮
2. 选择一个测评（如MBTI）
3. 阅读测评介绍，点击开始
4. 完成所有问题（至少5道）
5. 提交测评
6. 查看结果
7. 点击分享按钮
8. 尝试导出功能
9. 返回首页

**预期结果:**
- 每个步骤都能正常跳转
- 答题过程流畅
- 结果页面完整展示
- 分享/导出功能可用

### TC-002: 随机测评抽查
**测试步骤:**
1. 进入测评列表
2. 随机选择3个不同分类的测评
3. 每个测评完成5道题
4. 提交并查看结果

**预期结果:**
- 所有测评都能正常进行
- 评分系统正常工作
- 结果展示符合预期

### TC-003: 边界情况测试
**测试场景:**
1. 快速连续点击（测试防抖）
2. 网络中断后恢复
3. 刷新页面后状态保持
4. 后退按钮行为

**预期结果:**
- 系统不会崩溃
- 数据不会丢失
- 错误提示友好

## 自动化测试脚本

### 脚本1: 完整流程自动化
```python
# test_complete_flow.py
from playwright.sync_api import sync_playwright

def test_full_flow():
    with sync_playwright() as p:
        browser = p.chromium.launch(headless=True)
        page = browser.new_page()

        # 首页
        page.goto('http://localhost:5173')
        page.wait_for_load_state('networkidle')

        # 导航到测评列表
        page.click('text=测评')
        page.wait_for_load_state('networkidle')

        # 选择第一个测评
        page.click('.assessment-card >> nth=0')
        page.wait_for_load_state('networkidle')

        # 开始测评
        page.click('button:has-text("开始")')
        page.wait_for_load_state('networkidle')

        # 回答5道题
        for i in range(5):
            page.click('.option >> nth=0')
            page.click('button:has-text("下一")')
            page.wait_for_timeout(500)

        # 提交
        page.click('button:has-text("提交")')
        page.wait_for_load_state('networkidle')

        # 验证结果页面
        assert page.locator('.result-chart').is_visible()

        browser.close()
```

```python
def test_cross_browser():
    browsers = [
        ('chromium', 'Chrome'),
        ('firefox', 'Firefox'),
    ]

    for browser_type, browser_name in browsers:
        with sync_playwright() as p:
            browser = getattr(p, browser_type).launch()
            page = browser.new_page()
            page.goto('http://localhost:5173')
            # ... 测试步骤
            print(f"✓ {browser_name} 测试通过")
            browser.close()
```

### 脚本3: 性能基准测试
```python
def test_performance():
    import time

    with sync_playwright() as p:
        browser = p.chromium.launch()
        page = browser.new_page()

        start_time = time.time()
        page.goto('http://localhost:5173')
        page.wait_for_load_state('networkidle')
        load_time = time.time() - start_time

        print(f"首页加载时间: {load_time:.2f}秒")
        assert load_time < 3.0, f"加载时间过长: {load_time:.2f}秒"

        browser.close()
```

## 风险和缓解

### 风险1: 外部依赖不稳定
**影响:** 测试结果可能受影响
**缓解:** 使用mock数据或本地资源

### 风险2: 动态内容导致测试不稳定
**影响:** 测试可能出现随机失败
**缓解:** 增加适当的等待时间，使用稳定的选择器

### 风险3: 不同环境表现不一致
**影响:** 本地通过但生产失败
**缓解:** 在接近生产环境的环境中测试
